TTTStats

A website displaying statistics about Trouble in Terrorist Town played with some friends.

The website is written in Angular.js and accesses the RESTTT Backend for the displayed information.

See here

RESTTT Backend

The backend for TTTStats, implemented as REST API with Express.

It fetches logfiles from the game server via FTP, stores them in the cloud. These files are parsed using regexes and a callback-based system to populate a MySQL database with high-level information. Finally, several routes offer access to data computed through SQL queries.
